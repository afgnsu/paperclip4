#■Paperclip 4 Lab - 蘇介吾 105/05/13

##功能：讓上傳照片用ImageMagick產生五種檔案格式，並用年月當目錄排序

1. original => Default Size
2. public   => 640x640
3. view     => 360x360
4. cover    => 180x180
5. thumb    => 80x80
